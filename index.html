Introduction

AnticipatoRy Complex Adaptive Network Extrapolation for Apache Spark (ARCANE-Spark) version 1.0.0 is a Scala-based Apache Spark binding for ARCANE. ARCANE is a genetic algorithm system for automatically generating system dynamics models that differ in both their embedded parameters and their fundamental equations while maintaining dimensional consistency. ARCANE is intended to be used for anticipatory modeling of systems such as complex networks. The system can be activated using a one line call:

    ARCANEEngine.run("local", true, "input//matrixscenario_1", 5)

The first parameter is a string for the Apache Spark URL or the kind of run to complete. The second parameter is a boolean for the use the population size as read from the Matrix Engine file in the URL. The third parameter is the path to the Matrix Engine file to load. The fourth parameter is the number of genetic algorithm steps to execute. A reference for the underlying ARCANE API can be found here. The license can be found here.
