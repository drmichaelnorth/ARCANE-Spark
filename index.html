Introduction

AnticipatoRy Complex Adaptive Network Extrapolation for Apache Spark (ARCANE-Spark) version 1.0.0 is a Scala-based Apache Spark binding for ARCANE. ARCANE is a genetic algorithm system for automatically generating system dynamics models that differ in both their embedded parameters and their fundamental equations while maintaining dimensional consistency. ARCANE is intended to be used for anticipatory modeling of systems such as complex networks.
